LOOP Creator OS MVP（Windows）

双击 LOOP-Creator-OS-MVP.exe 即可启动中文工作台。

使用条件：
1. EXE 必须保留在本项目的 ui/release 目录中；
2. Windows 已安装 Node.js 22.13 或更高版本；
3. 首次启动会自动安装缺失的 npm 依赖，可能需要几分钟；
4. 工作台会提示配置邮件池；只有 AI 个性化模式才需要模型 API Key；
5. 右下角托盘图标可重新打开工作台、查看日志或停止本次启动的服务。

这是未签名的 MVP 内部测试程序。Windows 可能显示 SmartScreen 提示；生产分发前应使用公司的代码签名证书签名。
